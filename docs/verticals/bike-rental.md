# Bike/Scooter Rental Vertical

## Industry Overview

### **Market Size**
- **Vietnam:** ~5,000 bike/scooter rental businesses
- **Da Nang:** ~150 businesses (tourist hotspot)
- **Average daily rentals:** 10-50 bikes per business
- **Tourist market:** 18M tourists/year pre-COVID

### **Current Pain Points**
1. **Manual booking process** (WhatsApp/Zalo chaos)
2. **No real-time availability tracking**
3. **Document collection nightmare** (passport photocopies)
4. **Paper rental agreements** (legal risk)
5. **No payment integration** (cash only = security risk)
6. **Poor online presence** (Facebook page only)
7. **Language barriers** with international tourists
8. **No damage reporting system**

## Technical Implementation

### **Database Schema**

```sql
-- Fleet table
CREATE TABLE bike_fleet (
  id UUID PRIMARY KEY,
  hub_id UUID REFERENCES hubs(id),
  model_name VARCHAR(100),
  brand VARCHAR(50),
  engine_size INTEGER, -- cc
  transmission VARCHAR(10), -- 'auto' or 'manual'
  year INTEGER,
  color VARCHAR(30),
  license_plate VARCHAR(20) UNIQUE,
  photos JSONB, -- array of image URLs
  daily_rate INTEGER, -- VND
  is_available BOOLEAN DEFAULT true,
  condition_notes TEXT,
  mileage INTEGER,
  last_maintenance_date DATE,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Bookings table
CREATE TABLE bike_bookings (
  id UUID PRIMARY KEY,
  bike_id UUID REFERENCES bike_fleet(id),
  hub_id UUID REFERENCES hubs(id),
  customer_name VARCHAR(100),
  customer_phone VARCHAR(20),
  customer_email VARCHAR(100),
  customer_id_photo_url TEXT, -- passport/ID upload
  customer_selfie_url TEXT, -- face verification
  pickup_date DATE,
  return_date DATE,
  pickup_location VARCHAR(10), -- 'shop' or 'delivery'
  delivery_address TEXT,
  delivery_fee INTEGER,
  total_days INTEGER,
  base_price INTEGER,
  addons JSONB, -- [{"item": "helmet", "price": 50000}]
  discount_percent INTEGER DEFAULT 0,
  total_price INTEGER,
  payment_method VARCHAR(20),
  payment_status VARCHAR(20), -- 'pending', 'paid', 'refunded'
  booking_status VARCHAR(20), -- 'confirmed', 'active', 'completed', 'cancelled'
  contract_signed_at TIMESTAMP,
  contract_pdf_url TEXT,
  notes TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Damage reports
CREATE TABLE damage_reports (
  id UUID PRIMARY KEY,
  booking_id UUID REFERENCES bike_bookings(id),
  bike_id UUID REFERENCES bike_fleet(id),
  report_type VARCHAR(20), -- 'pre_rental', 'post_rental'
  photos JSONB, -- array of damage photos
  description TEXT,
  damage_cost INTEGER, -- VND
  insurance_covered BOOLEAN,
  created_at TIMESTAMP DEFAULT NOW()
);
```
